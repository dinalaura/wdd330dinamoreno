W02 Notes from JavaScript: Novice to Ninja

This article is based on personal notes from chapters 2, 3, and 4 from Javascript novice to ninja.

Programming Basic

In programming, a comment is a piece of code that is ignored by the language. It makes it easier for
anybody to read code.

// this is a short comment

/* This is a longer comment */

The JavaScript syntax is C-style because of its similarities with the C programming language.
Statements end with a new line or semicolon; there's no need to use a semicolon because interpreters
use Automatic Semicolon Insertion; however, it is a good practice.

const message = "Hello World!";

A block is a series of statements that are collected together inside curly braces;
blocks don't need to be terminated by a semicolon. Whitespace in the code makes it neat and easy to read.

There are reserved words for the use of JavaScript. Those words are not available to name variables,
functions, or objects. There are few words though that should be reserved but are not:
undefined
NaN
Infinity

JavaScript has seven different data types. Six of them are primitive data types:
String
Symbol
Number
Boolean
Undefined
Null

Objects are not primitive data, and these include arrays, functions, and object literals.
You can use typeof for finding the type of value, for example:
typeof "hello"
<< "string"

Variables are used in programming languages to refer to a value stored in memory,
there are several ways to declare a variable, and to assign a value we use the = operator.

const name = "Hello" // can't change
let name = "Hello" // can change

You can assign multiple variables at the same time:
let x=3, y=4, z=5

You should try to use const to declare most variables to avoid bugs.

In previous versions of JavaScript variables were declared with var. The difference between var and const
or let is that the last two have block scope.

Block scoped means that those variables only exits inside the block. Scope refers to where a constant or
variable is accessible by the program.

Global scope means that it is accessible everywhere in the program, and is not considered a good practice,
try to use it to a minimum.

Local scope use let or const, those are defined inside the block.

You should name constants and variables with sensible names that describe what the variable represents.
The names can start with any upper or lower-case letter, an underscore, or dollar character; they can
contain numbers but not start with them. Also, remember that variable names are case sensitive.

If you're using multiple words you may want to use Camel Case, it starts with lowercase and each new word
starts capitalized. Another option is to use underscores to separate but is not that convenient.

A string is a set of characters, letters, and symbols. You can create strings using a constructor function:

new String("hello")
<< [String: "hello"]

If you want to escape a special character you may want to use the backslash.

Primitive data types and objects have properties and methods. Properties are information about the object or value,
and methods perform an action on the object.

Dot notation is used to access the properties of a string.
name.length // variable's length property

A method is a task primitive data can perform, methods, and with parentheses. For example:

toUpperCase() method
name.toUpperCase();
<< "NAME"

There are a variety of methods:
toLowerCase();
charAt();
indexOf();
includes();
startWith();
concat();
trim();

Template literals use the backtick character ` to deliminate the string. They allow interpolation,
which is an expression that can be inserted inside a string.

const name = `Siri`; `Hello ${ name }!`; << 'Hello Siri!'

Symbols are a new primitive value in ES6. They create unique values, they don't have a literal form.
const uniqueID = Symbol();

Numbers can be integers or floating-point numbers. If a number starts with 0x is hexadecimal.
JavaScript allows exponential notation 1e6 // 1 multiplied by 10 to the power 6

Numbers also have methods:
toExponential()
toFixed()
toPrecision()

Arithmetic Operations can be carried out in JavaScript. Addition, subtraction, multiplication, division,
exponentiation, or a remainder of a division.

Arrays, Logic, and Loops

An array is an ordered list of values, to create an array literal write a pair of square brackets:
const myArray = [];

As I stated before arrays are not primitive values. Arrays use the index to access a specific value in an array.
heroes[0] = "Superman";

Each item in an array works as a variable, you change the value using the assignment operator.
Unsued slots in the array are filled by the value undefined. Arrays contain a variety of data types and
empty array objects.
const mixedArray = [null, 1, [], "two, true"];

The delete operator will remove an item from an array, but doesn't remove the place in the index,
that is replaced with undefined.

There are different properties and methods for arrays. The length property or pop(), push(),
shift() and unshift() method, etc. Also, you can find if an array contains a particular value
using the indexOf() method. Multidimensional arrays are arrays of arrays.

A set is a data structure that represents a collection of unique values, an empty set is creating
using the new operator:
const list = new Set();
You can add value:
list.add(1). Seth also has methods like the size() method or the has() method, this last one
checks if a value is in a set. To remove a value from a set use the delete() method.

Maps are another data structure, they are similar to objects, but they are limited to using
strings for key values, and other differences.

An empty map object uses the new operator and the Map() constructor.
const romanNumerals = new Map();
// add entries
romanNumeral.set(1, "I");
Some of the Map methods are get(), has(), delete() or clear().

Logic

If Statements are logical conditions that allow controlling the flow of a program.
if(condition){
// code to run if condition is true
}

The else keyword can be used to add an alternative block of code to run.
if (condition){
// code to run if condition is true}
else{
//code to run if condition is false}

The switch operator can be used to make code easier to read or follow.
switch (number) {
case 4:
console.log('You rolled a four'); break;
case 5:
console.log('You rolled a five'); break;

They are different kinds of loops, that is code that will repeat over and over again
according to conditions. While loops, do... while loops, for loops, nested for loops,
and also you have to take care of infinite loops.

Functions

Functions are first-class objects, they behave the same way as all the other primitive data types.
function hello(){
console.log("Hello World!")}

Invoking a function is to run the code inside of it.
hello();

All functions return a value, they use the word return as a statement.
A function that does not return anything will return undefined.

Any parameters a function needs are set when the function is defined.
When a function is invoked it is provided with arguments.

An arrow function is written like this:
const square = x => x*x;
const hello = () => alert("Hello World!");

A function that is passed as an argument to another is a callback:
function sing(song,callback) { console.log(`I'm singing along to ${song}.`); callback();
}
